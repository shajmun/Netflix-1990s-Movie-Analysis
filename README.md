# Netflix 1990s Movie Analysis

## Project Overview

This project is part of Associate Data Scienctist Track by Datacamp. This project performs an exploratory data analysis on the Netflix dataset, with a specific focus on movies released in the 1990s. The goal is to uncover insights about movie durations and genres from that nostalgic decade. This was a project to practice and showcase my data analysis skills using Python and pandas.

---

## Dataset

The dataset used is `netflix_data.csv`, which contains information about TV shows and movies available on Netflix. The key columns used for this analysis include:

* `type`: The type of content (Movie or TV Show).
* `title`: The title of the content.
* `release_year`: The year the content was released.
* `duration`: The duration of the movie in minutes.
* `genre`: The genre of the content.

---

## Tools and Libraries

* **Python:** The core programming language used for the analysis.
* **Pandas:** For data manipulation and analysis.
* **Jupyter Notebook:** For interactive coding and documentation.

---

## Analysis and Findings

The analysis focused on answering two main questions about movies from the 1990s:

1.  **What was the most frequent movie duration in the 1990s?**
    * **Answer:** The most common duration for a movie in the 1990s was **`94`** minutes.

2.  **How many short action movies were released in the 1990s?**
    * (A movie is considered "short" if it's less than 90 minutes.)
    * **Answer:** There were **`7`** short action movies released in the 1990s.
